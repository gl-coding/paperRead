🎨 彩色标注功能 - 使用指南
================================

✨ 全新功能：用6种颜色标注单词！
===============================

📋 功能介绍：
-----------
现在你可以使用6种不同颜色来标注单词，让学习更有条理！
不同颜色可以代表不同的含义，帮助你更好地组织和记忆词汇。

🎨 6种颜色及建议用途：
-------------------

🟡 黄色 (#ffc107)
  建议用途：标记生词、不认识的单词
  记忆提示：黄色像提醒灯，提醒你这是需要学习的词

🟢 绿色 (#28a745) [默认]
  建议用途：重点词汇、核心概念
  记忆提示：绿色代表通过，这些是要掌握的重点

🟠 橙色 (#ff5722)
  建议用途：需要复习的单词
  记忆提示：橙色像警告，提醒你需要加强记忆

🔵 蓝色 (#2196f3)
  建议用途：短语搭配、固定用法
  记忆提示：蓝色冷静，适合标记需要整体记忆的内容

🟣 紫色 (#9c27b0)
  建议用途：高级词汇、学术用语
  记忆提示：紫色高贵，适合标记高级表达

🔴 红色 (#f44336)
  建议用途：易错词、需特别注意的词
  记忆提示：红色警示，这些词容易搞混或用错

🖱️ 使用方法：
-----------

步骤1: 开启标注模式
  → 在侧边栏找到"📝 标注模式"
  → 勾选复选框
  → 颜色选择器会自动展开

步骤2: 选择颜色
  → 点击其中一个颜色圆按钮
  → 被选中的颜色会有黑色边框
  → 绿色是默认选中的颜色

步骤3: 标注单词
  → 在文章中点击要标注的单词
  → 单词会变成选中的颜色
  → 文章中所有相同单词都会被标注

步骤4: 更换颜色标注
  → 点击选择其他颜色
  → 继续点击单词进行标注
  → 可以用不同颜色标注不同的单词

步骤5: 取消标注
  → 再次点击已标注的单词
  → 该单词的标注会被取消
  → 文章中所有相同单词的标注都会消失

步骤6: 清除所有标注
  → 点击"清除标注 (N个单词)"按钮
  → 所有颜色标注都会被清除

步骤7: 退出标注模式
  → 取消勾选"标注模式"复选框
  → 颜色选择器会隐藏
  → 回到普通高亮模式
  → 已标注的单词会保留标注

💡 实用场景：
-----------

场景1: 首次阅读文章
  1. 开启标注模式，选择黄色
  2. 阅读时遇到生词就点击标注
  3. 完成后看看标注了多少生词
  4. 集中学习这些黄色标注的词

场景2: 标记不同类型词汇
  1. 黄色标注：生词
  2. 绿色标注：核心词汇
  3. 蓝色标注：固定搭配
  4. 紫色标注：学术词汇
  5. 一目了然地看到词汇分布

场景3: 复习巩固
  1. 第一天：黄色标注所有生词（假设20个）
  2. 学习后退出标注模式
  3. 第二天：开启标注模式，用橙色标注还不熟的词（假设10个）
  4. 第三天：用红色标注最难记的词（假设3个）
  5. 清楚看到学习进度

场景4: 词汇分类学习
  → 绿色：标注所有动词
  → 蓝色：标注所有名词
  → 紫色：标注所有形容词
  → 分类掌握不同词性

场景5: 难度分级
  → 黄色：基础词汇
  → 橙色：中级词汇
  → 红色：高级词汇
  → 紫色：专业术语

🎯 学习流程示例：
---------------

第一遍阅读（标记生词）：
  1. 开启标注模式
  2. 选择黄色
  3. 阅读文章，遇到生词就点击
  4. 标注完成，查看总共多少生词
  5. 关闭标注模式，查阅词典学习

第二遍阅读（标记重点）：
  1. 开启标注模式
  2. 选择绿色
  3. 标注文章的核心词汇和关键概念
  4. 绿色和黄色标注共存
  5. 重点词汇一目了然

第三遍阅读（标记短语）：
  1. 开启标注模式
  2. 选择蓝色
  3. 标注固定搭配和短语
  4. 三种颜色形成完整的学习体系

第四遍复习（标记难点）：
  1. 开启标注模式
  2. 选择红色
  3. 标注还记不住或易错的词
  4. 红色标注是最需要关注的部分

定期回顾：
  1. 打开之前读过的文章
  2. 看到彩色标注
  3. 回忆每个词的意思
  4. 不熟的可以重新标注橙色

📊 侧边栏联动：
-------------

标注效果：
  → 标注的单词在侧边栏会显示对应颜色
  → 背景色变为标注颜色
  → 文字变为白色
  → 左侧有颜色边栏

快速查看：
  → 在侧边栏快速找到不同颜色的标注
  → 按频率排序，看哪些标注词出现最多
  → 按A-Z排序，系统化复习标注词

侧边栏标注：
  → 在标注模式下，可以在侧边栏点击单词
  → 效果和在文章中点击一样
  → 更适合快速批量标注

🎓 高级技巧：
-----------

技巧1: 颜色编码系统
  为自己创建一套颜色编码规则：
  - 黄 = 完全不认识
  - 绿 = 见过但不确定
  - 蓝 = 认识但不会用
  - 紫 = 会用但不熟练
  - 红 = 容易混淆

技巧2: 主题标注
  同一篇文章读多次，每次关注不同主题：
  - 第一次：标注科技词汇（绿色）
  - 第二次：标注商业词汇（蓝色）
  - 第三次：标注情感词汇（紫色）

技巧3: 难度递进
  - 橙色：中考/高考词汇
  - 绿色：四六级词汇
  - 蓝色：考研词汇
  - 紫色：GRE/托福词汇
  - 红色：专业术语

技巧4: 记忆阶段
  - 黄色：第一次见（需要查词典）
  - 橙色：看过一次（有点印象）
  - 绿色：看过两次（基本记住）
  - 蓝色：看过三次（已掌握）
  - 紫色：完全掌握（可以使用）

技巧5: 错误类型
  - 红色：拼写易错
  - 橙色：发音易错
  - 紫色：用法易错
  - 蓝色：搭配易错

⚙️ 标注模式切换：
---------------

普通模式 → 标注模式：
  1. 勾选"标注模式"
  2. 颜色选择器展开
  3. 点击单词会标注
  4. 之前的标注会保留

标注模式 → 普通模式：
  1. 取消勾选"标注模式"
  2. 颜色选择器隐藏
  3. 点击单词会黄色高亮
  4. 彩色标注依然显示但不可编辑

💾 注意事项：
-----------

✓ 标注会在当前会话中保留
✓ 切换文章时标注会自动清除
✓ 刷新页面后标注会丢失
✓ 每个单词只能有一种颜色标注
✓ 重新标注会覆盖之前的颜色
✓ 标注和普通高亮可以共存

✗ 标注不会自动保存到文件
✗ 无法导出标注列表（未来功能）
✗ 无法在不同文章间共享标注

🌟 最佳实践：
-----------

1. 建立自己的颜色系统，保持一致
2. 不要标注太多，重点是重要的词
3. 定期清除已掌握词汇的标注
4. 配合翻译功能使用效果更好
5. 善用侧边栏的颜色视觉效果

4. 点击几个单词试试
5. 切换不同颜色标注更多单词
6. 观察侧边栏的颜色显示
