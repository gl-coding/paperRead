📖 英文文章阅读器 - 使用说明
================================

🎉 新功能：中文翻译已添加！

✨ 主要功能：
-----------

1. 📝 文章阅读
   - 在文本框中粘贴或输入英文文章
   - 或点击"加载示例文章"按钮体验

2. 🌏 中文翻译（新！）
   - 点击"🌏 显示中文翻译"按钮
   - 系统会自动调用翻译API
   - 中文翻译显示在对应英文段落下方
   - 再次点击按钮可隐藏翻译
   - 翻译结果会被缓存，重复查看更快
   - 蓝色渐变背景，清晰易读

3. 📊 单词统计
   - 自动提取所有单词
   - 显示每个单词的出现频率
   - 统计总词数和不同单词数

4. 🔍 单词筛选
   - 全部：按原始顺序
   - A-Z排序：字母顺序
   - 按频率：从高到低

5. 🎯 交互功能
   - 点击文章中的单词：高亮显示所有相同单词（黄色）
   - 按住 Ctrl/⌘ 点击：多选单词标注（绿色）
   - 点击侧边栏单词：文章中对应单词高亮
   - 清除选择按钮：一键取消所有多选标注
   - 搜索框：快速查找单词

🚀 快速开始：
-----------

1. 双击打开 index.html 文件
2. 点击"加载示例文章"按钮
3. 点击"🌏 显示中文翻译"，中文会出现在英文下方
4. 再次点击按钮可隐藏翻译（按钮显示"✓ 隐藏翻译"）
5. 在侧边栏浏览单词列表
6. 直接点击单词：单选高亮（黄色）
7. 按住 Ctrl/⌘ 点击：多选标注（绿色）
8. 点击"清除选择"按钮取消所有标注

💡 使用建议：
-----------

✓ 先尝试阅读英文原文
✓ 遇到困难时再查看中文翻译（对照学习）
✓ 点击隐藏翻译，测试自己是否理解
✓ 通过单词列表掌握高频词汇
✓ 利用高亮功能学习词汇在上下文中的用法
✓ 使用多选功能标注多个生词，便于集中复习

⚠️ 注意事项：
-----------

• 翻译功能需要网络连接
• 使用免费API，每天有调用次数限制
• 如果翻译失败，可以：
  - 使用浏览器内置翻译功能
  - 复制文本到其他翻译工具
  - 稍后重试
