🎯 多选单词标注功能 - 使用指南
================================

✨ 新功能：同时标注多个单词
===========================

📋 功能说明：
-----------
现在你可以同时选中和标注多个单词，方便标记生词、
重点词汇或需要特别关注的单词，非常适合学习场景！

🎨 视觉区分：
-----------

单选模式（默认）：
  颜色：黄色背景
  操作：直接点击单词
  效果：之前的选择会被清除

多选模式：
  颜色：绿色背景
  操作：按住 Ctrl (Mac 用 ⌘) + 点击单词
  效果：累积选择，不会清除之前的选择

侧边栏显示：
  多选的单词：绿色背景
  计数显示：清除选择按钮会显示已选数量

🖱️ 操作方法：
-----------

方法1: 文章中多选
  1. 按住 Ctrl 键（Mac 按 ⌘）
  2. 点击第一个单词 → 变绿色
  3. 继续按住 Ctrl，点击第二个单词 → 也变绿色
  4. 重复操作，可以选中更多单词
  5. 再次点击已选中的单词 → 取消选中

方法2: 侧边栏多选
  1. 在侧边栏找到目标单词
  2. 按住 Ctrl 键（Mac 按 ⌘）
  3. 依次点击多个单词
  4. 文章中对应单词都会变绿色高亮

方法3: 清除所有选择
  1. 选中多个单词后，会出现"清除选择"按钮
  2. 按钮上显示已选单词数量
  3. 点击按钮，一键清除所有标注

💡 使用场景：
-----------

场景1: 标记生词
  → 阅读时遇到不认识的单词
  → 按住 Ctrl 点击标记（绿色）
  → 继续阅读，遇到其他生词继续标记
  → 阅读完后，所有生词都被标注出来
  → 集中查询和学习这些生词

场景2: 标注重点词汇
  → 找出文章中的关键术语
  → 多选标注所有关键词
  → 清晰看到这些词在文章中的分布
  → 理解文章的主题和结构

场景3: 对比学习
  → 选中多个同义词或相关词
  → 观察它们在不同语境中的用法
  → 理解细微的差别

场景4: 词频分析
  → 标注你想重点学习的单词
  → 在侧边栏看到它们的频率
  → 优先学习高频词汇

🎓 学习流程示例：
---------------

步骤1: 第一遍阅读
  → 正常阅读文章
  → 遇到生词，按 Ctrl + 点击标记
  → 不中断阅读节奏，继续往下

步骤2: 统计生词
  → 阅读完毕，查看绿色标注
  → "清除选择"按钮显示生词总数
  → 评估文章难度

步骤3: 集中学习
  → 在侧边栏浏览绿色标注的生词
  → 点击每个生词，查看在文章中的位置
  → 结合上下文理解词义

步骤4: 验证学习
  → 点击"清除选择"按钮
  → 再读一遍文章
  → 遇到刚学过的词，思考含义
  → 如果还不确定，再次标注

步骤5: 反复巩固
  → 过几天重读文章
  → 标注还记不住的词
  → 观察生词数量是否减少

⌨️ 快捷键说明：
-------------

Windows/Linux:
  Ctrl + 点击单词 = 多选模式
  直接点击单词 = 单选模式（清除其他选择）

Mac:
  ⌘ (Command) + 点击单词 = 多选模式
  直接点击单词 = 单选模式（清除其他选择）

🎯 高级技巧：
-----------

技巧1: 快速标记同类词
  → 在侧边栏按住 Ctrl
  → 快速点击多个相关单词
  → 比在文章中逐个找更高效

技巧2: 分组学习
  → 第一轮：标记名词
  → 清除，第二轮：标记动词
  → 清除，第三轮：标记形容词
  → 分类掌握词汇

技巧3: 渐进式学习
  → 第一天：标记所有生词（可能10个）
  → 学习后清除
  → 第二天：重新阅读，标记还不熟的（可能5个）
  → 第三天：再次验证（可能2个）
  → 直到不需要标注

技巧4: 词汇笔记
  → 选中本文的重点词汇
  → 截图保存（带绿色标注）
  → 作为学习笔记保存

🌟 实用组合：
-----------

组合1: 多选 + 翻译
  1. 阅读英文原文
  2. 多选标注生词
  3. 显示中文翻译
  4. 对照学习生词的意思

组合2: 多选 + 频率排序
  1. 标注目标词汇

  3. 看到标注词的出现次数
  4. 优先学习高频词

组合3: 多选 + 搜索

  3. 在搜索结果中添加新的标注
  4. 建立个性化词汇表
